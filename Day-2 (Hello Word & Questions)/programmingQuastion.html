1. What is programming?

Programming jisme computer ko instructions diye jaate hain taaki wo specific tasks perform kar sake.

2. What is a translator in programming?

Translator (jo hum likhte hain) ko machine code (0s aur 1s) me convert karta hai, taki computer usko samajh sake aur execute kar sake.

3. Types of translators in C and C++?

Translator ke Do Types hote hai 1st Compiler 2nd Interpreter.

4. Difference between Compiler and Interpreter?

Compiler pura code ek sath translate karta hai.
Interpreter line-by-line translate karta hai.
